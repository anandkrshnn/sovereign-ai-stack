# 🗺️ Sovereign AI Stack Roadmap (v1.1.0 & Beyond)

This roadmap outlines the strategic direction for the Sovereign AI Stack as we transition from v1.0.0-GA to enterprise-wide adoption.

## 🏁 v1.0.0-GA (CURRENT)
**Focus: The Trinity of Trust (Retrieve. Verify. Gate. Prove.)**
- [x] **Verified Airlock**: Mandatory grounding verification for all RAG answers.
- [x] **Local Forensic Anchor**: Hardware-bound audit chain integrity via OS Secure Enclave.
- [x] **Unified Identity**: Consistent principal attributes across Bridge, RAG, and Agent.
- [x] **Consolidated Monorepo**: Unified `sovereign-ai` package.
- [x] **OpenAI-Compatible Gateway**: Multi-tenant bridge with `/v1/chat/completions`.
- [x] **Observability Core**: Prometheus `/metrics` and OTLP Tracing integrated into the Bridge.

---

## 🚀 v1.1.0 (Q2 2026)
**Focus: Developer Experience & Operational Non-Repudiation**
- [ ] **Remote Forensic Anchors**: Production-ready backends for Git, IPFS, and Blockchain non-repudiation.
- [ ] **Docker & K8s Orchestration**: Official Helm charts and Docker Compose for air-gapped k8s clusters.
- [ ] **Encryption-at-Rest (Hardened)**: Migration to SQLCipher4 for all local knowledge vaults.
- [ ] **Sovereign Cookbooks**: Comprehensive documentation with "Day-1" examples for Healthcare and Finance.
- [ ] **Enhanced observability**: Pre-built Grafana dashboards for the Sovereign Gateway.

---

## 🏗️ v1.2.0 (Q3 2026)
**Focus: Ecosystem & Multi-Model Intelligence**
- [ ] **Multi-Model Orchestration**: Dynamic routing between local models (Llama 3, Qwen, Mistral) based on intent.
- [ ] **LangChain / LlamaIndex Integration**: Native providers for the Sovereign Stack.
- [ ] **Web Admin UI**: React-based dashboard for tenant provisioning and policy management.
- [ ] **Rate Limiting & Quotas**: Enterprise-grade usage control per principal.

---

## 🌍 Long-term Vision (v2.0+)
- [ ] **Distributed Forensics**: Federated audit chains across multiple edge nodes.
- [ ] **Hardware Security Modules (HSM)**: Integration with FIPS 140-2 Level 3 hardware.
- [ ] **Agentic Compliance**: Autonomous agents that monitor and enforce GAIP-2030 standards in real-time.

---

## 💼 Commercial Roadmap
- [ ] **Enterprise Pilot Program**: 2-week "Airlock" implementation for regulated teams.
- [ ] **Managed Sovereign Cloud**: For customers who want the stack's guarantees on managed bare-metal.
- [ ] **Compliance Certifications**: Mapping forensic trails to SOC 2 Type II and HIPAA attestations.
