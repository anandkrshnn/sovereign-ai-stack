# Contributing to the Sovereign AI Stack

Thank you for your interest in contributing!

## How to Contribute

### Report Bugs
Open a [New Issue](https://github.com/anandkrshnn/sovereign-ai-stack/issues/new) with:
- A clear title
- Steps to reproduce
- Expected vs actual behavior

### Suggest Features
Start a [Discussion](https://github.com/anandkrshnn/sovereign-ai-stack/discussions) before opening a PR.

### Submit Pull Requests
1. Fork the repository
2. Create a branch: `git checkout -b feat/your-feature`
3. Make changes with clear commit messages
4. Open a PR against `main`

### Improve Documentation
Doc fixes and clarifications are always welcome — open a PR directly.

## Security Issues
Email [ananda.krishnan@hotmail.com](mailto:ananda.krishnan@hotmail.com) — do not open a public issue.

## Code of Conduct
Be respectful and constructive. See [protocol-z-federate/CODE_OF_CONDUCT.md](https://github.com/anandkrshnn/protocol-z-federate/blob/main/CODE_OF_CONDUCT.md).
